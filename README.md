# Analiza-zadovoljstva-putnika-u-zracnom-prometu
Podaci preuzeti sa kaggle.com i generirani preko generatedata.com, očistiti će se i pripremiti za daljnju analizu. Analizirati će se zadovoljstvo putnika u zračnom prometu te stres i broj članova obitelji koji putuju sa putnikom. Rezultati koji se dobiju prikazati će se preko grafova.
