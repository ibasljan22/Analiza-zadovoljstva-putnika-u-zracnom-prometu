# Analiza-zadovoljstva-putnika-u-zracnom-prometu
CSV datoteka sa podacima preuzeta sa kaggle.com, a JSON datoteka ima generirane podatke preko generatedata.com. Podaci se čiste i pripremaju za daljnju analizu. Analizirati će se zadovoljstvo putnika u zračnom prometu s obzirom na razinu stresa, vrsti klase na zrakoplovu i tipu kupca. Rezultati koji se dobiju prikazati će se preko grafova.
